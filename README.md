A web-based career guidance platform that helps students explore suitable career paths through secure authentication and personalized career recommendations.

## 📖 About the Project

Career Compass is a web-based career guidance platform designed to help students discover suitable career paths according to their interest. The application provides secure authentication, personalized dashboards, and career recommendations through a clean and responsive interface.
